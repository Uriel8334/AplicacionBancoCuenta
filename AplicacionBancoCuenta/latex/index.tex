\chapter{Sistema de Gestion Bancaria }
\hypertarget{index}{}\label{index}\index{Sistema de Gestion Bancaria@{Sistema de Gestion Bancaria}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduccion}{Introduccion}}\label{index_intro_sec}
El Sistema de Gestion Bancaria es una aplicacion de consola desarrollada en C++ que simula las operaciones basicas de un banco. El sistema permite gestionar clientes, cuentas bancarias (ahorro y corriente), realizar transacciones y generar reportes.

La aplicacion utiliza conceptos avanzados de programacion orientada a objetos como herencia, polimorfismo, templates y estructuras de datos personalizadas.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Caracteristicas Principales}{Caracteristicas Principales}}\label{index_features_sec}

\begin{DoxyItemize}
\item Gestion de clientes (creacion, busqueda, actualizacion)
\item Administracion de cuentas de ahorro y cuentas corrientes
\item Operaciones bancarias (depositos, retiros, transferencias)
\item Validacion de datos y fechas
\item Persistencia de datos mediante archivos
\item Interfaz de texto con marquesina informativa
\item Generacion de codigos QR para cuentas
\item Implementacion didactica de estructuras de datos (listas enlazadas, arbol B)
\item \doxylink{class_cifrado}{Cifrado} y descifrado de informacion
\end{DoxyItemize}\hypertarget{index_modules_sec}{}\doxysection{\texorpdfstring{Modulos Principales}{Modulos Principales}}\label{index_modules_sec}
\hypertarget{index_cliente_mod}{}\doxysubsection{\texorpdfstring{Gestion de Clientes}{Gestion de Clientes}}\label{index_cliente_mod}
Este modulo se encarga de administrar la informacion personal de los clientes del banco. La clase principal es \doxylink{class_persona}{Persona} que almacena datos como nombre, apellido, direccion, etc.\hypertarget{index_cuentas_mod}{}\doxysubsection{\texorpdfstring{Gestion de Cuentas}{Gestion de Cuentas}}\label{index_cuentas_mod}
Modulo para administrar las cuentas bancarias. La clase base es \doxylink{class_cuenta}{Cuenta}, de la cual heredan \doxylink{class_cuenta_ahorros}{Cuenta\+Ahorros} y \doxylink{class_cuenta_corriente}{Cuenta\+Corriente}, cada una con sus propias reglas de negocio.\hypertarget{index_util_mod}{}\doxysubsection{\texorpdfstring{Utilidades}{Utilidades}}\label{index_util_mod}
