\chapter{Sistema de Gestión Bancaria }
\hypertarget{index}{}\label{index}\index{Sistema de Gestión Bancaria@{Sistema de Gestión Bancaria}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introducción}{Introducción}}\label{index_intro_sec}
El Sistema de Gestión Bancaria es una aplicación de consola desarrollada en C++ que simula las operaciones básicas de un banco. El sistema permite gestionar clientes, cuentas bancarias (ahorro y corriente), realizar transacciones y generar reportes.

La aplicación utiliza conceptos avanzados de programación orientada a objetos como herencia, polimorfismo, templates y estructuras de datos personalizadas.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Características Principales}{Características Principales}}\label{index_features_sec}

\begin{DoxyItemize}
\item Gestión de clientes (creación, búsqueda, actualización)
\item Administración de cuentas de ahorro y cuentas corrientes
\item Operaciones bancarias (depósitos, retiros, transferencias)
\item Validación de datos y fechas
\item Persistencia de datos mediante archivos
\item Interfaz de texto con marquesina informativa
\item Generación de códigos QR para cuentas
\item Implementación didáctica de estructuras de datos (listas enlazadas, árbol B)
\item \doxylink{class_cifrado}{Cifrado} y descifrado de información
\end{DoxyItemize}\hypertarget{index_modules_sec}{}\doxysection{\texorpdfstring{Módulos Principales}{Módulos Principales}}\label{index_modules_sec}
\hypertarget{index_cliente_mod}{}\doxysubsection{\texorpdfstring{Gestión de Clientes}{Gestión de Clientes}}\label{index_cliente_mod}
