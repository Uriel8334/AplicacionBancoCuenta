\doxysubsection{Detailed Description}
Sistema de gestion bancaria con interfaz de consola. 

Este archivo implementa la aplicacion principal del sistema bancario, proporcionando un menu interactivo para gestionar cuentas, realizar transferencias, buscar informacion, y generar reportes. Incluye una marquesina en la parte superior para mensajes importantes.
