\doxysubsection{Detailed Description}
Clase que representa un banco con sus operaciones principales. 

Esta clase gestiona la lista de personas (clientes) del banco y proporciona funcionalidades para crear, buscar y administrar cuentas bancarias. Tambien permite realizar operaciones como transferencias y manejo de archivos para persistencia de datos.
