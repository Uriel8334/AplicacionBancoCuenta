\doxysubsection{Detailed Description}
Clase que implementa un nodo para una lista enlazada de personas. 

Esta clase forma parte de una estructura de datos tipo lista enlazada donde cada nodo contiene un puntero a un objeto \doxylink{class_persona}{Persona} y un puntero al siguiente nodo en la lista.
