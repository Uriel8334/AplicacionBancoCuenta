\doxysubsection{Detailed Description}
Implementacion de una marquesina de texto desplazable para consola. 

Esta clase permite mostrar un texto que se desplaza horizontalmente en una consola de Windows, con soporte para colores y formato. Los datos se pueden cargar desde un archivo HTML y se actualizan automaticamente cuando el archivo cambia.
