\doxysubsection{Detailed Description}
Espacio de nombres que contiene las clases y funciones para generacion de codigos QR. 

Este namespace proporciona una implementacion completa para generar, manipular y visualizar codigos QR segun el estandar ISO/\+IEC 18004.
