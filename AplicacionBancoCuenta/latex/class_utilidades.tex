\doxysubsection{Detailed Description}
Clase estatica con funciones auxiliares para todo el sistema bancario. 

Esta clase proporciona metodos utilitarios para manipulacion de datos, formateo, validaciones, operaciones con archivos, generacion de codigos QR, y otras funcionalidades de soporte que son utilizadas por todo el sistema bancario.
