\doxysubsection{Detailed Description}
Clase que implementa funcionalidades de cifrado y descifrado de archivos. 

Esta clase estatica proporciona metodos para cifrar y descifrar archivos utilizando el algoritmo XOR. Tambien contiene funcionalidades especificas para el manejo de datos del banco, permitiendo cifrar, guardar, y recuperar informacion bancaria de manera segura.
