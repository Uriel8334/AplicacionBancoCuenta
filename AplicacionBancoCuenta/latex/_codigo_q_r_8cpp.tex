\doxysubsection{Detailed Description}
Implementacion de las clases Qr\+Segment y Qr\+Code para generacion de codigos QR. 

Este archivo contiene la implementacion completa de la biblioteca para generar codigos QR. Incluye algoritmos para codificacion de datos, correccion de errores Reed-\/\+Solomon y generacion de la matriz del codigo QR.
