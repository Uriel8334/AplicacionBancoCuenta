class Arbol\+B$<$ T $>$}
Implementación didáctica de un Árbol B para propósitos educativos. 

Esta clase implementa una versión simplificada de un Árbol B para ilustrar sus principios básicos de funcionamiento. Permite buscar elementos y visualizar la estructura del árbol.
