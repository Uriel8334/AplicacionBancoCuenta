Detailed Description

Clase que representa una cuenta de ahorros bancaria.

Esta clase hereda de la clase abstracta Cuenta<double> e implementa las funcionalidades específicas de una cuenta de ahorros, incluyendo el cálculo de intereses y operaciones básicas bancarias.
