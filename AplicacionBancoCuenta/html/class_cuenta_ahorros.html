Detailed Description

Clase que representa una cuenta de ahorros bancaria.

Esta clase hereda de la clase abstracta Cuenta<double> e implementa las funcionalidades especificas de una cuenta de ahorros, incluyendo el calculo de intereses y operaciones basicas bancarias.
