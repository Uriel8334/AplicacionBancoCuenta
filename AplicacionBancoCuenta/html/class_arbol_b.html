Detailed Description

template<typename T>
class ArbolB< T >

Implementacion didactica de un arbol B para propositos educativos.

Esta clase implementa una version simplificada de un arbol B para ilustrar sus principios basicos de funcionamiento. Permite buscar elementos y visualizar la estructura del arbol.
