Detailed Description

Clase que representa una cuenta corriente bancaria.

Esta clase hereda de la clase abstracta Cuenta<double> e implementa las funcionalidades específicas de una cuenta corriente, incluyendo la verificación de monto mínimo y operaciones bancarias básicas.
