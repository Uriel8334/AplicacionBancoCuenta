Detailed Description

Sistema de gestión bancaria con interfaz de consola.

Este archivo implementa la aplicación principal del sistema bancario, proporcionando un menú interactivo para gestionar cuentas, realizar transferencias, buscar información, y generar reportes. Incluye una marquesina en la parte superior para mensajes importantes.
