Detailed Description

Declaración adelantada (forward declaration) de la clase Persona.

Clase que representa a una persona cliente del banco.

Esta clase se declara aquí pero se define completamente en otro archivo. Contiene información personal de clientes y gestiona sus cuentas bancarias.

Esta clase gestiona la información personal de un cliente y administra sus cuentas bancarias (tanto de ahorros como corrientes). Implementa operaciones para crear, buscar y gestionar cuentas asociadas a la persona.
