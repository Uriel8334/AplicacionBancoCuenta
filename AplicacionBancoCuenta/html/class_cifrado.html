Detailed Description

Clase que implementa funcionalidades de cifrado y descifrado de archivos.

Esta clase estática proporciona métodos para cifrar y descifrar archivos utilizando el algoritmo XOR. También contiene funcionalidades específicas para el manejo de datos del banco, permitiendo cifrar, guardar, y recuperar información bancaria de manera segura.
